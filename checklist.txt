Roles:
1.Student
2.Admin
3.Faculty 
4.Exam Controller 
5.Club Coordinator 

Features of each role
i. Student:
-> Access to course and curriculum information.
-> Downloading of hall tickets.
-> Viewing circulars and notifications.
-> Access to club information.
-> Submitting faculty feedback.
-> Self-performance analysis and progress tracking.

ii. Admin:
-> Approval of fee payments and hall tickets.
-> Publishing circulars and notifications.
-> Designing and updating courses and curriculum.
-> Providing academic timetables.
-> Generating analytical reports for all user roles.

iii. Faculty:
-> Uploading study materials and assignments.
-> Conducting tests and evaluations.
-> Providing student remarks and feedback.
-> Analyzing student performance.
-> Posting and managing attendance records.
-> Showcasing achievements and previous academic work.
-> Preparation of key sheet.

iv. Exam Controller:
-> Reviewing and analyzing seating blueprints.
-> Allocating seating arrangements for students.
-> Generation of hall tickets.
-> Examination Notification.
-> Allocation of exam Evaluation for Faculty.

v. Club Coordinator:
-> Managing club member details.
-> Organizing and updating event information.
-> Approving event invitations.
-> Recording and displaying club achievements.
